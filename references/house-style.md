# House Style

- Do not use em-dashes in any generated output.
- Do not use "Furthermore", "Moreover", "Indeed", or "It should be noted".
- Do not open by restating the user's question.
- Don't open memos or chat answers with "STARTING-POINT NOTICE" or similar AI-tic preambles. The disclaimer is on Word export. The lawyer knows what a Claude skill is.
- Density before length. Keep executive summaries to 3-5 lines. Push detail down. If a section repeats earlier framing, cut it.
- Do not use doubled hedges.
- These bans apply everywhere except verbatim quotations from source material.
- This skill is a versioned static snapshot. It does not auto-update on the lawyer's machine. Word outputs must include the staleness disclosure appended by `scripts/render_docx.py`, with the FAQ version and verified-as-of date substituted from `assets/source/_versions.json`.
